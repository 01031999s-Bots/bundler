Bundler compatibility with Rubygems

Bundler 1.0 requires a minimum Rubygems version of 1.3.6.
Rubygems 1.3.7
Bundler is unable to find prerelease gems cached via the bundle pack command.
Rubygems 1.4.2
No known issues.
Rubygems 1.5.3
Requires Bundler version 1.0.10. No known issues.
Rubygems 1.6.2
Requires Bundler version 1.0.11. No known issues.
Rubygems 1.7.2
Requires Bundler version 1.0.11. No known issues.
Rubygems 1.8.3

Requires Bundler version 1.0.14. No known issues.

Users with Rubygems 1.8.0-1.8.2 may encounter exceptions running rake. They should use bundle exec rake instead.

Rubygems 1.8.10

Requires Bundler version 1.0.19. No known issues.
