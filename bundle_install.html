The best way to manage Ruby applications

bundle install

Make sure all dependencies in your Gemfile are available to your application. Gems may be installed to system gems or BUNDLE_PATH (defaults to ~/.bundle). Git repos will be checked out to BUNDLE_PATH. 
$ bundle install
Install your dependencies to a different location than BUNDLE_PATH. Further bundle commands or calls to Bundler.setup or Bundler.require will remember this location Learn More: Bundler.setup Learn More: Bundler.require 
$ bundle install /path
Install all dependencies, except those in groups you explicitly excluded Learn More: Groups 
$ bundle install --without development test
